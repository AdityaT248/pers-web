About This Website

This portfolio website was designed and developed by Aditya Thakkar in 2023. It was built using modern web technologies including:

Frontend

  • • HTML5 & CSS3
  • • Tailwind CSS for styling
  • • JavaScript for interactivity
  • • Responsive design for all devices

Backend

  • • Python with FastAPI framework
  • • Jinja2 templates
  • • RESTful API architecture
  • • Formspree for contact form handling

Deployment

  • • Containerized with Docker
  • • CI/CD pipeline for automated deployments
  • • Cloud hosting for reliability
  • • Version controlled with Git

The site features dark/light mode, responsive design, and is optimized for performance and accessibility. This project showcases my skills in full-stack development and my commitment to creating elegant, user-friendly web experiences.
